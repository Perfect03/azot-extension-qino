# Azot Qino Extension

<img src="https://qino.sequoia-api.lfstrm.tv/upload/9a88489434069a8c62ef9fc97bf7502d.svg" height="100" alt="Qino" />

This extension for [Azot](https://github.com/azot-labs/azot) lets you to download movies, series and shows from the [Qino](https://qino.live/) streaming service.
